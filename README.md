-AI Interview Practice Coach



Problem Statement:

Freshers struggle to practice interview questions and lack real-time feedback.



 Solution:

An AI-powered platform that conducts mock interviews and provides instant scoring and feedback.



Features:

\- Role-based interview questions

\- Answer evaluation with score

\- Improvement suggestions



Tech Stack:

\- Frontend: HTML, CSS, JavaScript

\- Backend: Python (Flask)

\- AI: LLM API

\- Hosting: Render



Architecture:

User → Web UI → Flask Backend → AI API → Feedback → UI



Deployment:
